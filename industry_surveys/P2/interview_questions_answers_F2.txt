Fault Related Questions:

Q: What's the appearance of the fault?
A: Some information displayed in a report is wrong.

Q: How to reproduce the fault?
A: When user access the report dashboard, some report are occasionally not correctly renderred.  

Q: How many microservices involved in this case?
A: 1.

Q: How much time spent to locate the fault?
A: 6 days.

Q: What's the root cause of the fault?
A: Different data requests for the same report are returned in an unexpected order.

Q: Describe the detail process of the fault localication?
A: First check the ajax request one by one separately.
   Second check the backend handler for each ajax request.
   Third check the charts of the dashboard together in different combination.
   Finally check the chart rendering logic in different ajax returning sequence, and locate the issue.
